# Audit Archive

These reports preserve the evidence trail for the two remediation passes. They
are historical snapshots, not evergreen project documentation.

- `review-baseline.md`: baseline before the original Web client remediation.
- `final-review.md`: final report for the original remediation pass.
- `followup-baseline.md`: baseline before the concurrency, protocol, security,
  and deployment follow-up.
- `followup-final-review.md`: current follow-up report, including post-audit
  blocker closure and final verification evidence.

Use the newest follow-up report when deciding whether to merge the branch.
